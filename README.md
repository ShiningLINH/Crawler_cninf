# Crawler_cninf
基于Selenium的巨潮资讯网公司公告pdf批量下载

## 基本功能
1. 根据输入的目标公司名称（或者股票代码）列表，自动下载目标类型的公告（如：年报、季报...），并存放到不同文件夹中（一个公司对应一个文件夹） <br />
*p.s. 注释非常清晰！按需改动即可！*
